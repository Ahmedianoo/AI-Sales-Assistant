# ai-sales-assistant

This project is about creating an AI assistant for sales teams

First time running docker:
    docker-compose up --build
    
Use for day to day dev when we want to stop containers:
    docker-compose stop

optional if there r issues: 
    docker-compose build --no-cache
    docker-compose down
